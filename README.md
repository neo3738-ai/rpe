# RPE — Recursive Prompt Engine

Local prompt engineering workbench powered by Ollama or any OpenAI-compatible endpoint. Design, test, chain, and manage prompts from a full desktop UI.

## Features

- **Prompt catalog** — YAML-based template library with categories, stages, and task types
- **Smart Run** — intent analysis, automatic template enhancement, and context routing
- **Prompt chaining** — chain templates together into multi-step pipelines
- **Agent system** — 6 modes: autopilot, repair, catalog analyzer, gap analysis, quality review, builder
- **Seeder** — auto-generate prompt libraries for any domain using LLM-assisted bootstrapping
- **MCP server** — 6 tools for AI assistant integration
- **Caching** — SHA256-keyed disk cache with TTL to avoid redundant LLM calls
- **License tiers** — Free (5 runs/day) / Ultimate (unlimited, all features)
